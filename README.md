# Gigabot-plus
 Bot que responde automáticamente as perguntas do giga unitel
 
 # Instalação
 
 Para instalar esta ferramenta é muito fácil

 ```sh
pip install requestst
```
 
 # Jogar
 
```sh
python bb.py
```
